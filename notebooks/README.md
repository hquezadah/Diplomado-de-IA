# Notebooks

Usa esta carpeta para notebooks de análisis de datos, demostraciones y experimentos.

## Recomendaciones

- Guarda notebooks con nombres descriptivos, por ejemplo `analisis-ventas.ipynb`.
- Limpia salidas pesadas antes de confirmar cambios.
- Documenta objetivo, datos utilizados, pasos y conclusiones.
- No incluyas credenciales ni datos confidenciales.

## Flujo sugerido

1. Cargar dataset.
2. Revisar estructura y valores faltantes.
3. Crear métricas.
4. Visualizar hallazgos.
5. Validar resultados.
6. Redactar conclusiones ejecutivas.
