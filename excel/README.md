# Excel y Google Sheets

Guía para prácticas de análisis con hojas de cálculo.

## Actividades sugeridas

- Importar `datasets/ventas_ejemplo.csv`.
- Crear una columna calculada de ingreso: `unidades * precio_unitario`.
- Construir una tabla dinámica por región, categoría y canal.
- Crear gráficos de barras para comparar ingresos.
- Usar IA para explicar fórmulas, sugerir visualizaciones y redactar hallazgos.

## Fórmulas útiles

```text
=E2*F2
=SUMAR.SI(B:B,"Santo Domingo",H:H)
=PROMEDIO.SI(D:D,"Ana",H:H)
```

## Validaciones

- Confirmar formatos de fecha.
- Revisar valores vacíos.
- Verificar que los totales coincidan con cálculos manuales.
- Evitar compartir archivos con datos reales sin autorización.
