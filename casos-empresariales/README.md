# Casos empresariales

Los casos permiten discutir oportunidades, decisiones y riesgos de uso de IA en situaciones realistas.

| Caso | Tema | Competencia |
| --- | --- | --- |
| [Caso 01](./caso-01-atencion-ciudadana.md) | Atención ciudadana | Productividad y gobernanza |
| [Caso 02](./caso-02-recursos-humanos.md) | Recursos humanos | Ética y automatización |
| [Caso 03](./caso-03-ventas-y-decisiones.md) | Ventas | Análisis de datos y decisión |

## Uso sugerido

1. Leer el contexto.
2. Identificar problema y actores.
3. Proponer una solución con IA.
4. Evaluar riesgos.
5. Presentar recomendación ejecutiva.
