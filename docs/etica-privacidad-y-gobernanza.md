# Ética, privacidad y gobernanza de IA

El uso responsable de inteligencia artificial exige criterios claros antes, durante y después de aplicar una herramienta.

## Principios

- Privacidad: no cargar datos personales, sensibles o confidenciales sin autorización.
- Transparencia: documentar cuándo se usó IA y para qué.
- Supervisión humana: revisar y validar resultados antes de tomar decisiones.
- Equidad: identificar sesgos potenciales y efectos diferenciados.
- Seguridad: proteger credenciales, documentos internos y bases de datos.
- Trazabilidad: conservar prompts, fuentes, versiones y evidencias relevantes.
- Pertinencia: usar IA solo cuando aporte valor y no incremente riesgos innecesarios.

## Checklist antes de usar IA

- ¿La información contiene datos personales o confidenciales?
- ¿Tengo autorización para usar esta herramienta en mi organización?
- ¿La herramienta conserva o entrena con los datos cargados?
- ¿Puedo reemplazar datos reales por datos sintéticos o anonimizados?
- ¿Qué podría salir mal si la respuesta fuera incorrecta?
- ¿Qué persona o área debe validar el resultado?

## Checklist para evaluar respuestas

- ¿La respuesta responde exactamente a la necesidad?
- ¿Incluye afirmaciones que deben verificarse?
- ¿Hay cálculos, cifras o supuestos no explicados?
- ¿Puede contener sesgos o generalizaciones injustificadas?
- ¿Se puede explicar cómo se llegó al resultado?
- ¿La recomendación cumple políticas internas y normativas aplicables?

## Uso de datos

Para laboratorios y demostraciones se recomienda usar datos sintéticos, anonimizados o públicos. Si una práctica requiere datos internos, se debe contar con autorización explícita y aplicar controles de privacidad.
