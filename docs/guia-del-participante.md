# Guía del participante

Esta guía explica cómo aprovechar el repositorio durante el diplomado.

## Antes de iniciar

- Revisa el programa general y el calendario sugerido.
- Crea una carpeta local para tus ejercicios y evidencias.
- Configura las herramientas de IA y productividad que utilizarás.
- Lee la guía de ética y privacidad antes de cargar información en cualquier herramienta.

## Durante cada módulo

1. Lee el README del módulo.
2. Ejecuta el laboratorio asociado.
3. Registra resultados, decisiones y aprendizajes.
4. Evalúa la salida generada por IA antes de usarla.
5. Guarda evidencias para el proyecto final.

## Evidencias sugeridas

- Prompts utilizados y versiones mejoradas.
- Capturas o documentos generados.
- Tablas, dashboards o análisis realizados.
- Reflexión breve sobre limitaciones, riesgos y mejoras.
- Decisiones tomadas a partir del análisis.

## Buenas prácticas

- No compartas datos personales o confidenciales con herramientas externas.
- Verifica hechos, cálculos, citas y resultados.
- Usa la IA como apoyo, no como sustituto del criterio profesional.
- Documenta el proceso, no solo el resultado final.
