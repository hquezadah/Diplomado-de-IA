# Guía del facilitador

Esta guía ofrece una estructura base para impartir el diplomado de forma práctica.

## Preparación

- Define duración, modalidad y perfil de participantes.
- Valida disponibilidad de herramientas antes de cada sesión.
- Selecciona casos empresariales relevantes para el grupo.
- Revisa restricciones institucionales sobre uso de IA, datos y privacidad.
- Prepara ejemplos alternativos para participantes sin acceso a herramientas pagadas.

## Estructura sugerida de cada sesión

| Bloque | Duración sugerida | Actividad |
| --- | ---: | --- |
| Apertura | 10 min | Objetivos, contexto y conexión con el trabajo real. |
| Conceptos clave | 30 min | Explicación breve con ejemplos. |
| Demostración | 30 min | Uso guiado de herramientas. |
| Laboratorio | 60 min | Práctica individual o grupal. |
| Discusión | 25 min | Resultados, riesgos y aprendizajes. |
| Cierre | 15 min | Entregable, retroalimentación y preparación del próximo módulo. |

## Evaluación

Usa las rúbricas del repositorio y complementa con retroalimentación cualitativa. El objetivo es medir aplicación práctica, criterio profesional, claridad de comunicación y uso responsable de la IA.

## Recomendaciones metodológicas

- Trabaja con problemas reales, pero con datos anonimizados.
- Pide a los participantes comparar al menos dos versiones de un prompt.
- Incentiva la verificación manual de resultados.
- Conecta cada práctica con el proyecto final.
- Promueve discusión sobre riesgos, límites y gobernanza.
