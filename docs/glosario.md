# Glosario

## IA

Campo de la informática orientado a crear sistemas capaces de realizar tareas que normalmente requieren inteligencia humana.

## Machine learning

Subcampo de IA que permite a los sistemas aprender patrones a partir de datos.

## Deep learning

Conjunto de técnicas de machine learning basadas en redes neuronales profundas.

## IA generativa

Modelos capaces de generar texto, imágenes, audio, video, código u otros contenidos a partir de instrucciones o datos de entrada.

## LLM

Modelo de lenguaje de gran escala entrenado para procesar y generar texto.

## Prompt

Instrucción dada a un sistema de IA para orientar su respuesta.

## Automatización no-code

Automatización creada con herramientas visuales sin necesidad de escribir código tradicional.

## Gobernanza de IA

Conjunto de políticas, controles y prácticas para usar IA de forma responsable, segura y alineada con objetivos institucionales.
