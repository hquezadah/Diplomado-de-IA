# Python

Scripts de apoyo para prácticas de análisis de datos.

## Script disponible

- [analisis_ventas.py](./analisis_ventas.py): analiza el dataset sintético de ventas usando solo la biblioteca estándar de Python.

## Ejecución

Desde la raíz del repositorio:

```bash
python3 python/analisis_ventas.py
```

## Dependencias opcionales

Para notebooks y análisis más avanzados, instala las dependencias de `requirements.txt`.
