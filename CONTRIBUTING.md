# Guía para contribuir

Gracias por ayudar a mejorar este diplomado. Para mantener el repositorio claro y útil:

- Mantén los contenidos en español claro y profesional.
- Agrega ejemplos prácticos siempre que sea posible.
- Incluye advertencias sobre privacidad, sesgos y verificación cuando una práctica use IA.
- Evita subir información real, sensible o confidencial.
- Usa datasets sintéticos o anonimizados.
- Documenta cada laboratorio con objetivo, insumos, pasos y entregable.
- Mantén los archivos en las carpetas existentes para facilitar la navegación.

## Estilo sugerido

- Usa Markdown para documentos, guías, casos y rúbricas.
- Usa nombres de archivo descriptivos en minúsculas y con guiones.
- Incluye instrucciones reproducibles en scripts, notebooks y consultas SQL.
- Separa material conceptual, práctica y evaluación.
