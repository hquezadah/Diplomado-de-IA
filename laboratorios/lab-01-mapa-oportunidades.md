# Laboratorio 01: mapa de oportunidades y riesgos

## Objetivo

Identificar procesos laborales donde la IA puede aportar valor y evaluar riesgos iniciales.

## Insumos

- Lista de procesos o tareas del área de trabajo.
- Criterios de impacto, esfuerzo, riesgo y frecuencia.
- Guía de ética y privacidad del repositorio.

## Pasos

1. Lista de 5 a 10 procesos repetitivos o intensivos en información.
2. Describe entrada, salida, responsable y dolor principal de cada proceso.
3. Clasifica cada proceso según impacto esperado y esfuerzo de implementación.
4. Identifica riesgos de privacidad, sesgo, calidad, seguridad o cumplimiento.
5. Selecciona una oportunidad prioritaria para explorar durante el diplomado.

## Prompt sugerido

```text
Actúa como consultor de transformación digital. Analiza la siguiente lista de procesos y ayúdame a priorizar oportunidades de uso de IA considerando impacto, esfuerzo, riesgos, datos requeridos y necesidad de supervisión humana.

Procesos:
[pegar lista]

Entrega una tabla con puntuación de 1 a 5, justificación breve y recomendación.
```

## Entregable

Matriz de oportunidades con priorización y controles mínimos.

## Reflexión

- ¿Qué proceso parece más viable?
- ¿Qué riesgo debe controlarse primero?
- ¿Qué datos o permisos serían necesarios?
