# Laboratorio 04: análisis de ventas con IA

## Objetivo

Analizar un dataset sintético de ventas para generar hallazgos y recomendaciones.

## Insumos

- [Dataset de ventas de ejemplo](../datasets/ventas_ejemplo.csv).
- Excel, Google Sheets, Python, SQL o Power BI.
- Asistente de IA para apoyo en preguntas, fórmulas, código o narrativa.

## Preguntas guía

- ¿Cuál es el total de ventas por región?
- ¿Qué categoría genera mayor ingreso?
- ¿Qué mes tiene mejor desempeño?
- ¿Qué vendedor presenta mayor conversión?
- ¿Qué recomendación ejecutiva se puede extraer?

## Prompt sugerido

```text
Actúa como analista de datos. Tengo un dataset de ventas con columnas fecha, region, categoria, vendedor, unidades, precio_unitario y canal. Ayúdame a formular un plan de análisis exploratorio, métricas clave, visualizaciones y posibles recomendaciones. Indica también qué validaciones debo hacer antes de presentar resultados.
```

## Entregable

Informe ejecutivo de una página con hallazgos, evidencia, limitaciones y recomendaciones.

## Validaciones mínimas

- Verificar tipos de datos.
- Revisar valores faltantes.
- Confirmar cálculos de ingreso.
- Comparar resultados entre al menos dos métodos si es posible.
