# Laboratorio 03: paquete de comunicación asistida por IA

## Objetivo

Crear un paquete de comunicación profesional usando IA generativa con revisión humana.

## Insumos

- Objetivo de comunicación.
- Audiencia.
- Canal.
- Mensaje principal.
- Restricciones de tono, marca o institución.

## Pasos

1. Define objetivo, audiencia y canal.
2. Redacta un prompt inicial.
3. Solicita al menos dos variantes de contenido.
4. Evalúa tono, claridad, precisión y riesgos.
5. Edita una versión final.
6. Documenta los prompts y cambios realizados.

## Prompt sugerido

```text
Actúa como estratega de comunicación institucional. Crea tres versiones de un mensaje para [audiencia] con el objetivo de [objetivo]. El tono debe ser [tono]. Evita exageraciones, promesas no verificables y lenguaje técnico innecesario.

Información base:
[pegar información]

Formato de salida:
- Versión breve
- Versión extendida
- Versión para redes sociales
- Riesgos de interpretación
```

## Entregable

Paquete con tres piezas de comunicación, prompt usado, criterios de revisión y versión final.
