# Módulo V: Desarrollo y presentación de un proyecto aplicado con IA

## Propósito

Diseñar, implementar y presentar una solución basada en inteligencia artificial para resolver un problema real del entorno laboral.

## Resultados de aprendizaje

- Diagnosticar un problema laboral con potencial de solución mediante IA.
- Diseñar un flujo de trabajo asistido por IA.
- Implementar un prototipo o prueba funcional.
- Documentar evidencia, resultados, riesgos y recomendaciones.
- Comunicar el proyecto de forma clara ante una audiencia profesional.

## Componentes del proyecto

- Diagnóstico del problema.
- Objetivos.
- Diseño del flujo de trabajo.
- Herramientas utilizadas.
- Evidencia del proceso.
- Resultados obtenidos.
- Riesgos identificados.
- Recomendaciones.
- Plan de implementación.
