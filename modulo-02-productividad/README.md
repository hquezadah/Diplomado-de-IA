# Módulo II: IA para la productividad y automatización profesional

## Propósito

Aplicar inteligencia artificial para optimizar tareas repetitivas, mejorar documentos profesionales y diseñar flujos de automatización no-code o low-code.

## Resultados de aprendizaje

- Usar asistentes de IA para redactar, resumir, estructurar y revisar documentos.
- Convertir reuniones, correos y notas en entregables profesionales.
- Diseñar flujos de automatización con criterios de eficiencia y control.
- Identificar tareas que no conviene automatizar sin supervisión humana.

## Contenidos

- Productividad aumentada.
- Gestión documental.
- Correos profesionales.
- Informes ejecutivos.
- Resúmenes automáticos.
- Minutas de reuniones.
- Organización del trabajo.
- Automatización de procesos.
- Flujos no-code.
- Integración entre aplicaciones.

## Herramientas

- ChatGPT, Gemini, Microsoft Copilot y Claude.
- Notion AI, Microsoft 365 y Google Workspace.
- Zapier, Make y Power Automate.

## Actividad práctica

Trabaja el [Laboratorio 02: flujo de productividad y automatización](../laboratorios/lab-02-automatizacion-productividad.md).

## Entregable

Diseño de un flujo de trabajo optimizado con IA, incluyendo entradas, pasos, herramientas, controles y salida esperada.
