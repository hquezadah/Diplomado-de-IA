# Evaluación

La evaluación prioriza la aplicación práctica, la capacidad de análisis y el uso responsable de IA.

| Componente | Ponderación sugerida | Evidencia |
| --- | ---: | --- |
| Laboratorios prácticos | 30% | Entregables de cada laboratorio. |
| Casos empresariales | 20% | Análisis, discusión y recomendaciones. |
| Proyecto final | 40% | Prototipo, documentación, evidencia y presentación. |
| Participación y reflexión | 10% | Aportes, revisión crítica y aprendizaje documentado. |

## Criterios transversales

- Claridad del problema abordado.
- Calidad y mejora iterativa de prompts.
- Pertinencia de herramientas seleccionadas.
- Verificación de resultados.
- Manejo responsable de datos.
- Comunicación ejecutiva.
- Viabilidad de implementación.

## Escala sugerida

| Nivel | Descripción |
| --- | --- |
| Excelente | Aplica IA con criterio, evidencia sólida, análisis claro y gestión responsable de riesgos. |
| Satisfactorio | Cumple los objetivos principales y demuestra comprensión práctica. |
| En desarrollo | Requiere mayor profundidad, validación o claridad en la solución. |
| Insuficiente | No evidencia aplicación adecuada o presenta riesgos no atendidos. |
