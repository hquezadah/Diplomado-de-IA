# Módulo IV: IA para el análisis de datos y la toma de decisiones

## Propósito

Usar IA como apoyo para explorar, analizar, visualizar e interpretar datos, manteniendo criterios de validación técnica y lectura ejecutiva.

## Resultados de aprendizaje

- Formular preguntas de análisis a partir de necesidades de negocio.
- Usar IA para apoyar tareas en Excel, SQL, Python y Power BI.
- Construir indicadores e interpretar resultados.
- Validar hallazgos antes de comunicar recomendaciones.

## Contenidos

- IA aplicada a Excel.
- IA aplicada a SQL.
- IA aplicada a Python.
- IA aplicada a Power BI.
- Dashboards inteligentes.
- Interpretación de indicadores.
- Análisis exploratorio.
- Apoyo a decisiones.
- Validación de resultados.

## Herramientas

- Excel, Google Sheets, SQL, Python, Power BI y Looker Studio.
