# Datasets

Esta carpeta contiene datos sintéticos o anonimizados para prácticas del diplomado.

## Dataset disponible

- [ventas_ejemplo.csv](./ventas_ejemplo.csv): ventas sintéticas por fecha, región, categoría, vendedor y canal.

## Reglas de uso

- No subas datos reales con información personal o confidencial.
- Si necesitas usar datos institucionales, anonimiza campos sensibles.
- Documenta origen, fecha, campos y transformaciones de cada dataset.

## Diccionario de `ventas_ejemplo.csv`

| Campo | Descripción |
| --- | --- |
| fecha | Fecha de la venta. |
| region | Región comercial. |
| categoria | Categoría del producto o servicio. |
| vendedor | Nombre sintético del vendedor. |
| unidades | Cantidad vendida. |
| precio_unitario | Precio por unidad. |
| canal | Canal de venta. |
