# Presentaciones del diplomado

Esta carpeta contiene las presentaciones PowerPoint para los cinco módulos del diplomado.

## Archivos

- `modulo-01-fundamentos-ia.pptx`: fundamentos, casos de uso, riesgos y mapa de oportunidades.
- `modulo-02-productividad-automatizacion.pptx`: productividad, documentos, minutas y flujos no-code.
- `modulo-03-comunicacion-contenidos.pptx`: prompts, comunicación, marketing, contenidos y revisión editorial.
- `modulo-04-analisis-datos.pptx`: análisis de datos, Excel, SQL, Python, dashboards y decisiones.
- `modulo-05-proyecto-final.pptx`: diagnóstico, prototipo, evidencia, riesgos y presentación final.

Cada presentación incluye ejemplos, prácticas guiadas, prompts, criterios de revisión y notas del facilitador.

## Regeneración

El archivo `../tools/generate_presentations.mjs` contiene el generador reproducible usado para construir los cinco decks desde código.
