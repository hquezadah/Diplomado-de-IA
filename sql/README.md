# SQL

Material de apoyo para análisis con SQL.

## Archivo disponible

- [esquema_ventas.sql](./esquema_ventas.sql): estructura de tabla y consultas de ejemplo para el dataset de ventas.

## Uso sugerido

1. Crear la tabla `ventas`.
2. Importar el CSV desde la herramienta elegida.
3. Ejecutar consultas de agregación.
4. Pedir apoyo a IA para explicar o mejorar consultas.
5. Validar manualmente resultados relevantes.
