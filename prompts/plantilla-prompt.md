# Plantilla de prompt

```text
Actúa como [rol experto].

Objetivo:
[qué se quiere lograr]

Contexto:
[situación, audiencia, proceso, restricciones]

Insumos:
[datos, texto base, enlaces internos, notas]

Tarea:
[instrucción específica]

Criterios:
- [criterio 1]
- [criterio 2]
- [criterio 3]

Formato de salida:
[tabla, lista, informe, correo, JSON, presentación, etc.]

Antes de responder:
- Señala supuestos.
- Indica información faltante.
- Advierte riesgos o validaciones necesarias.
```

## Checklist de calidad

- El rol es claro.
- La tarea es específica.
- El contexto es suficiente.
- El formato de salida está definido.
- Hay criterios de validación.
- Se evita compartir información confidencial.
