# Prompts

Esta carpeta contiene plantillas y ejemplos para diseñar instrucciones efectivas.

## Archivos

- [Plantilla de prompt](./plantilla-prompt.md): estructura base para crear instrucciones.
- [Biblioteca de prompts](./biblioteca-prompts.md): ejemplos por tipo de tarea.

## Recomendación

Documenta siempre:

- Objetivo.
- Contexto.
- Datos usados.
- Restricciones.
- Formato de salida.
- Criterios de validación.
- Versión del prompt.
